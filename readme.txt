Woo-commerce Minimum Price Amount is an awesome minimum price setting plugin. Just need to install and set the minimum amount 
 
== Description ==
 A very very simple plugin to set minumum price of every woo-commerce order. In this plugin you can set minimum price for every order. When a customer want to check out form your woo-commerce shop he/She will get notification if the total price is below the plugin minimum price.

**Current Languages**

* Englishs.
 
== Installation ==
 

* Install from the wordpress plugin manager or upload the directory to your plugins folder
* Activate
* Navigate to woo-commerce setting -> minimum price and fill in the required settings
